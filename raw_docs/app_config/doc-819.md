表情标签配置说明

除了文字与语音播报，AIUI极速超拟人交互系统还提供了丰富的表情表达，**表情标签 (Facial Expression)**代表了 AI 回复时的情绪与内心状态，让开发者和用户能够感知到 AI 的“心情”与“意图”，从而为产品带来拟人灵动的交互体验。目前表情的颗粒度是会话级别的，即一轮AI回复带一个表情标签。

注意：该能力只能在**极速超拟人交互链路**下开启使用。

# 表情标签的用途

- 增强聊天沉浸感：在聊天气泡旁展示对应的 Emoji，根据情绪改变交互界面设计，让对话更有温度。
- 驱动面部表情：通过读取表情标签，可以让您的2D、3D 虚拟形象或机器人面部实时做出对应的表情（如大笑、哭泣、害羞），实现音容笑貌同步。
- 情绪分析统计：后台可以统计用户与 AI 交互过程中的情绪变化趋势，分析对话质量。

# 使用方式

如果您想申请接入表情驱动能力，请联系商务/技术支持获得授权。获得授权后，只需要在应用配置下打开表情驱动，即可获得云端下发的表情结果。下发数据结构示例：

```
{
'cbm_facial_expression': {
    'compress': 'raw',
    'encoding': 'utf8',
    'format': 'plain',
    'seq': 0,
    'status': 2,
    'text': 'happy'
    }
}
```
