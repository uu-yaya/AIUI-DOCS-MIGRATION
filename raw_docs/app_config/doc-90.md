结构化语义配置概述

结构化语义配置是传统应用配置技能的地方，支持配置的技能类型有：官方技能、自定义技能、语句问答、关键词问答、设备人设（注意区分这里的设备和极速链路回复角色区别，二者不是一个能力。前者是传统语义技能方式，后者是大模型prompt设置）；并且还支持进行特性设置，如自定义技能或自定义语句问答的阈值设置、关键词过滤、语义敏感词、精简协议。其中包括：

[**- 1、技能配置>>>点击跳转**](#技能配置)

[**- 2、自定义技能阈值设置>>>点击跳转**](#自定义技能阈值设置)

[**- 3、技能设置>>>点击跳转**](#技能设置)

## 技能配置

点击“添加”，可在当前应用的情景模式下配置需要的技能。

## 自定义技能阈值设置

在自定义技能添加界面，勾选自定义技能后，点击技能下的设置图标进行设置，注意支持的技能类型：
**自定义私有技能、自定义语句问答**
自定义阈值主要是为了做技能匹配度调节的，可以根据实际需要(模糊匹配度和误识别两个维度)综合考虑设置。
AIUI技能实际触发的模糊匹配程度，可以查看技能json中的source字段取值，可以参考该取值进行自定义技能阈值调整。

- 技能阈值越高：技能对出发的语句格式要求越严（模糊匹配越低，不容易误识别）
- 技能阈值越低：技能对出发的语句格式要求越送（模糊匹配越高，容易误识别）

## 技能设置

技能配置后，还可以点击“设置”进行一些高级能力配置，开发者按需设置。

- **关键词过滤**
  过滤识别结果中的关键词，不进行语义理解。常用于过滤唤醒词
   比如：唤醒词是“小飞小飞”
   **过滤前**：“小飞小飞，你好“
   **过滤后**：“你好“
- **语义敏感词过滤**
  给用户回复的信息中包含敏感信息，可以通过设置过滤掉。支持自定义增加。
  注意：
  当交互回复触发语义敏感词过滤时，固定响应：

  ```
  {
    answer: {
        text: "文明社会，请文明用语。",
        topic: "",
        type: "T"
    },
    operation: "ANSWER",
    rc: 0,
    service: "forbidden",
    sid: "xxxxxxxxx",
    state: null,
    text: "xxx",
    uuid: "xxxxxxxxx"
  }
  ```
- **精简协议**
  简化结构化语义，将技能结果分为：媒资类和播报类。默认关闭。
  将多个技能精简成一种输出协议，大大降低技能适配工作。适用于无屏设备。

![](/media/202508/2025-08-13_174431_6906150.07437879899617539.png "null")

![示例图片](/media/202508/2025-08-12_194246_5856210.3540283832065939.png)

注意当AIUI应用不开启**星火大模型配置**时，点击技能设置，还可以配置传统语义链路下的兜底闲聊。开启大模型时，可以由大模型做兜底回复，该能力被替代隐藏。
分别支持的类型有：

- 讯飞闲聊
- 讯飞视频搜索
- 无回复兜底
  该能力项下，开发者可以自定义回复语
