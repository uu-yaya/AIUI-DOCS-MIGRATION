## 1.文档问答说明

文档问答，是基于星火大模型知识库方案，高校检索文档信息，准确回答专业问题。支持开发者自定义知识点拆分逻辑、查看已拆分的知识点、编辑管理知识点。

上传文档支持两种方式：

- 本地文档
- 在线文档

## 文档问答分类

根据上传文档类型，主要分为一下两大类：

- 图文知识
- 表格知识

### 2.1. 图文知识

图文知识类文档问答，主要处理非表格文档，主持文档格式有：txt、doc、docx、pdf、md、xlsx。根据配置策略，图文知识也分为下面4种分类，详细为：

| 策略分类 | 策略说明 | 支持文档格式 |
| --- | --- | --- |
| 通用策略 | 通用智能，效果均衡。适用于合同、论文、行政文书等各类文档。 | 本地文档：txt、doc、docx、pdf、md  在线文档：网页链接 |
| 说明书策略 | 专为家电等领域的说明书设计，智能解析复杂排版与图文内容。 | 本地文档：txt、doc、docx、pdf、md  在线文档：网页链接 |
| Q-A对策略 | 高效解析Q-A对文档，适合问答清单。 | 本地文档：txt、doc、docx、xlsx |
| 长文本策略 | 专为10万字以上的文档设计，记忆关联远距离上下文信息。 | 本地文档：txt、doc、docx、pdf、md  在线文档：网页链接 |

### 2.2. 表格知识

表格知识类文档问答，主要是用来处理表格数据的。说明如下：

| 策略分类 | 策略说明 | 支持文档格式 |
| --- | --- | --- |
| 通用策略 | 智能解析Excel，支持多sheet，适用于单页单表、表头简单的表格。 | 本地文档：xlsx |

## 文档问答创建使用流程

当有本地或在线文档需要通过大模型加载使用时，可以创建文档问答来进行使用。

### 3.1. 创建问答库

在 “产品接入” 下的 “自定义业务” 页面中，点击”创建文档问答库“

![](/media/202410/1_1729221574.png "null")

![](/media/202410/2_1729221597.png "null")

根据需要选择对应的类别进行创建。

### 3.2. 文档上传或配置

- 本地文档上传
  进入文档问答配置页面，点击”文档管理“ ，在”本地文档”下点击”添加文档“，可以上传本地文档，注意文件格式要求说明。
  > 部分文档问答（图文知识下 Q-A策略类文档问答、表格知识文档问答）下上传文档支持模板参考，如不清楚具体格式，可以下载模板文档进行查看。

![](/media/202410/3_1729221625.png "null")

- 上传文档：本地文档选择加载，可以一次上传多个文档
  需要注意当前策略文档问答支持的文件格式、文件大小限制
- 设置”标签“和描述，非必填项，可忽略

![](/media/202410/4_1729221635.png "null")

- 在线文档配置

当创建的文档问答支持在线文档配置时，在文档管理页面，有“在线文档”选项。点击”在线文档”后点击”添加文档“，可配置需要加载的在线文档（设置的文档地址需支持外部公网访问）

![](/media/202410/5_1729221647.png "null")

- 在“网页地址”中填写文档url，多个文档换行添加
- 在“更新频率”中选择更新方式，支持 ：不更新、每日更新、每周更新、每月更新

  计算周期为：从设置更新频率开始计算周期
- 设置”标签“和描述，非必填项，可忽略

![](/media/202410/6_1729221655.png "null")

### 3.3. 文档信息设置

本地文档上传或在线文档配置后，支持操作有：

| 操作功能 | 说明 | 支持文档类别 |
| --- | --- | --- |
| 文件名称 | 自定义文档文件名称 | 本地文档、在线文档 |
| 更新频率 | 重新设置在线文档更新周期 | 在线文档 |
| 标签 | 自定义文档文件标签说明 | 本地文档、在线文档 |
| 是否生效 | 设置当前文档在本文档问答中是否使用 | 本地文档、在线文档 |
| 设置分段 | 设置文档问答知识点拆分策略 | 本地文档、在线文档 |
| 更新 | 立即更新在线文档加载 | 在线文档 |
| 下载原文 | 下载已上传的本地文档 | 本地文档 |
| 删除 | 删除当前选中的文档 | 本地文档、在线文档 |

![](/media/202410/7_1729221667.png "null")

![](/media/202410/8_1729221676.png "null")

其中“设置分段”主要提供有两种知识点拆分策略，开发者可以根据自己需要进行设置：

- 系统默认分段
- 自定义分段
  - 分段策略：标题分割、标识符分割
  - 标识符策略：
    - 分割符：遇到指定分隔符或文本超出最大分段长度时分段，不会丢失字符。支持自定义
    - 截断符：遇到截断符才分段。若两截断符之间的文本超过最大分段长度，将丢失超出的字符。支持自定义
  - 最大分段长度：[16,1024]字符区间设置

![](/media/202410/9_1729221686.png "null")

### 3.4. 知识点管理

在实际使用中，如果文档文档抽取的知识点不准确（多抽、少抽）、或无法问答、或不需要，可以通过知识点管理进行自定义编排处理。

![](/media/202410/10_1729221694.png "null")

- 查看：预览整个文档，查看当前知识点对应的问题和答案，可配置当前知识点是否使用，但不可修改知识点信息（关联问题和结果信息）
- 编辑：预览整个文档，查看当前知识点对应的问题和答案，可配置当前知识点是否使用，可修改知识点信息（关联问题和结果信息）
- 知识点配置开关：控制当前知识点是否配置使用

![](/media/202410/11_1729221703.png "null")

点击“构建发布”即可发布文档问答上线，发布成功后可以在AIUI应用下大模型下配置发布的文档问答。
