# 小程序配网

## 开机

连接USB供电，将电源开关拨到ON状态开机。

## 2.2. 进入配网模式

可以在设备未唤醒情况，直接说出离线识别命令词就可与套件进行交互。此模组默认支持以下离线识别命令词：

|  |  |
| --- | --- |
| **离线命令词** | **响应** |
| 配网模式 | 进入配网模式 |

## 2.3. 配网小程序

微信扫码登录”小飞陪伴”微信小程序。

![](/media/202509/2025-09-19_151628_1218800.993778014257793.png)

## 2.4. 配网步骤

- 步骤一：开发板上电，拨码开关开机后，语音喊：配网模式；
- 步骤二：微信打开”小飞陪伴”小程序；
- 步骤三：点击”设备”，添加设备；
- 步骤四：选择弹出的设备；
- 步骤五：输入wifi账号及密码；
- 步骤六：等待配网，配网成功开发板会有提示音。

# 交互方式

- 支持一次唤醒多次交互，交互停止60S后退出唤醒状态；
- 支持语音打断，全双工交互；
- 支持单双工交互切换；
- 支持按键打断当前对话。

# 固件更新说明

- 步骤一：准备一根typeC线束，typeC口接到7911主板，USB口连接电脑
- 步骤二：在烧录按键接口接上2pin1.25mm的按键（或者按键接口短接）
- 步骤三：按住烧录按键，同时打开拨码开关给主板上电（靠近模式切换按键处的灯亮红色），电脑会提示”叮咚”声进入烧录模式

![](/media/202509/2025-09-19_151828_9047920.9887597590541548.png)
