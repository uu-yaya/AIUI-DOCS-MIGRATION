# 产品概述

ZG803集成讯飞单麦离线识别算法，支持 3~5M 远场识别，支持唤醒词、离线识别命令词和本地回复语配置。

# 芯片硬件配置参数

**CPU：** 32-bit DSP，主频 160MHz
**SDRAM**：128KB
**Flash:** 512KB

# 产品功能

**语音唤醒**

- 安静场景唤醒率>95%
- 支持自定义唤醒词定制

**离线识别**

- 支持30条离线指令
- 支持自定义离线命令词
- 支持英、韩、日语、越南、印尼、西班牙等多语种识别

**串口通信**

- 支持串口下发指令，串口波特率921600

**多种发音人**

- 支持标准男生、女生等不同音色

**客制化**

- 可提供唤醒词定制工具，快速便捷实现定制

# 4.．唤醒词&离线命令词

1.默认唤醒词“小飞小飞”，客户可自定义唤醒词；
2.支持免唤醒交互，即在设备未唤醒情况，直接说出离线识别命令词就可以与模组进行交互
3.支持唤醒词+离线命令词的方式交互，即先说主唤醒词将设备唤醒，然后可以进行语音交互。

优先推荐采用唤醒词+离线命令词的方式交互，10 个以内命令词可采用免唤醒交互方式。
