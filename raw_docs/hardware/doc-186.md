# 产品概述

RK3328评估板套件是一站式语音交互解决方案，该套件包含RK3328 系统板 + 麦克风阵列板（线性四麦+线性六麦+环形六麦）。支持唤醒打断、全双工交互、UART 串口与上位机通信。可广泛应用在智能服务机器人、智能家居中控、自助服务终端等智能设备产品。
