背景介绍

本文档主要给开发者就目前创建的AIUI应用分类和授权方案做初步说明。

AIUI平台从2015年成立起，经过这么多年的服务更新和版本迭代，目前AIUI应用整体分两大类：

[**- 新应用：极速交互>>>点击跳转**](#新应用-极速交互)

[**- 老应用：传统语义和通用大模型>>>点击跳转**](#老应用)

所有应用都不限制接入方式：均可按照业务实际需要对接对应链路方案的SDK或API完成能力接入。

## 新应用-极速交互

### 1.1. 方案接入了解

👤该类应用属于AIUI最新极速超拟人交互方案绑定应用，新应用链路和授权方案固定，只能按
[**SDK开发接入 - 极速超拟人链路**](https://aiui-doc.xf-yun.com/project-1/doc-792/)
或
[**API开发接入 - 极速超拟人链路**](https://aiui-doc.xf-yun.com/project-1/doc-784/)方案接入

🔍关于极速超拟人语义服务与传统语义或通用大模型语义服务的区别，可以查看文档
[**AIUI服务链路**](https://aiui-doc.xf-yun.com/project-1/doc-403/)
详细了解方案接入。

### 1.2. 应用说明

极速交互类型，应用区分方式主要可以分为一下几点：

**- 创建时间**

从2025-06-12后创建的所有应用都属于极速交互类应用。

**- 应用配置**

应用下语义和大模型配置在服务链路中固定开启，不支持开发者自定义开关；另外新增基础配置设置，支持开发者进行识别引擎、兜底大模型、回复角色选配。

![](/media/202507/2025-07-28_161808_1761750.31918796616843326.png "null")

极速交互类应用，平台给与开发者的免费授权信息如下：

- 免费服务包：10000 次 有效期1年（从应用创建时计算）
- 默认API并发路数：3路

- 免费唤醒（VTN）装机量：20台

## 老应用

### 2.1. 方案接入了解

老应用按照配置不同，服务提供两种交互链路配置选型，分别为：

**- 传统语义链路**

按
[**SDK开发接入 - 传统语义链路**](https://aiui-doc.xf-yun.com/project-1/doc-790/)
或
[**API开发接入 - 传统语义链路**](https://aiui-doc.xf-yun.com/project-1/doc-782/)方案接入

**- 通用大模型链路**

按
[**SDK开发接入 - 通用大模型链路**](https://aiui-doc.xf-yun.com/project-1/doc-791/)
或
[**API开发接入 - 通用大模型链路**](https://aiui-doc.xf-yun.com/project-1/doc-783/)方案接入

关于传统语义和通用大模型语义服务与极速超拟人服务的区别，可以查看文档【[AIUI服务链路](https://aiui-doc.xf-yun.com/project-1/doc-403/ "AIUI服务链路")】详细了解。

### 2.2. 应用说明

老应用按时间周期也分两大类：

- 传统语义链路和通用大模型链路都支持
- 仅支持传统语义链路
  下面就两种不同应用做详细说明。

#### 2.2.1. 传统和大模型都支持的

该类应用是AIUI服务升级中间版本，开发者可以通过大模型开关来配置选择对应的服务链路：

- 大模型：开
  服务指向到通用大模型语义链路
- 大模型：关
  服务指向到传统语义链路。

![](/media/202507/2025-07-29_110713_8207170.39811918956504566.png "null")

应用区分方式：

- 创建时间
  从 **2022-06-01** 到 **2025-06-11** 时间周期内创建的所有应用。
- 功能模块
  传统语义、大模型、合成、后处理都支持模块化开关选配
- 应用信息
  应用右上角支持appid、appkey和apiserce信息查询

![](/media/202507/2025-07-29_111355_3778940.4990959468360413.png "null")

#### 2.2.2. 仅支持传统语义链路的

该类应用是AIUI最早应用版本，由于缺少秘钥信息应用不支持开启大模型，仅可以做传统语义方案对接。应用区分方式为：

- 创建时间
   至 **2022-06-01** 时间之前创建的所有应用。
- 功能模块
  - 传统语义、合成、后处理都支持模块化开关选配
  - 不支持大模型配置(开启会有协议不兼容提示)
- 应用信息
  仅提供appid和appkey，没有apisecret信息
