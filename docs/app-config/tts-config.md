---
title: 语音合成配置
---

::: info 概述
语音合成配置项是提供AIUI传统语义链路或通用大模型链路下，全链路语音合成服务开关和发音人选配。开启该配置项，服务请求时AIUI云服务就直接采用交互回复内容直接进行播报。

**使用流程**：

1. 登录 [AIUI 平台](https://aiui.xfyun.cn/) →「我的应用」
2. 选择目标应用 →「应用配置」
3. 找到「语音合成」配置项并开启/配置
:::

不同链路表现如下：

- **传统语义交互链路**
  - 未开启应用后处理：合成服务直接调用技能结果中 answer.text 字段取值直接合成。
  - 开启了应用后处理：合成服务直接调用后处理结果中 answer.text 字段取值直接合成。

- **通用大模型交互链路**：合成服务直接采用大模型结果的回复语进行合成（由于大模型结果流式下发，云端采用分段合成）

## 音色更换

在发音人下拉列表下可以选择需要的发音人选配，展示的发音人也可以通过筛选条件进行筛选。

- 场景
  - 全部场景
     **注意**：超拟人发音人在该场景下才能查看
  - 通用场景
  - 方言场景
  - 多语种场景
  - 民族语场景
- 年龄段
  - 全部年龄段
  - 女童声
  - 男童声
  - 成人女声
  - 成人男声

![](/media/202508/2025-08-14_145137_2225590.8225816224999526.png)

## 发音人授权查看

在发音人下拉列表下，黑色可选发音人代表已授权的，灰色代表未授权。具体表现如下：

- 黑色可选发音人
  - 永久免费发音人
  - 付费发音人
     注意：需查看真实授权的截止日期，可选不一定代表可用
- 灰色不可选发音人
   展示授权申请方式

## 发音人试听

该能力是提供给开发者在线试听发音人音色的，试听方式有两种：

- 在发音人选择界面直接点击播放按钮进行试听
- 选择发音人后自定义合成文本、调节合成参数进行合成试听
   注意：该方式仅支持可选发音人操作
