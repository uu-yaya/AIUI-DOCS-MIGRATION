title: AIUI 应用配置
---

::: info 概述
AIUI 应用配置是开发者在 [AIUI 平台](https://aiui.xfyun.cn/) 上为每个应用设置各项服务能力的入口。不同的配置项决定了应用可使用的语音识别、语义理解、合成、大模型、声纹、长时记忆等能力，也决定了服务链路的切换（传统语义 / 通用大模型 / 极速超拟人）。合理配置是实现产品预期体验的关键第一步。
:::

## 配置分类

AIUI 的应用配置按 **应用类型** 与 **配置通用性** 分为三大类：

- **通用配置**：不区分应用类型，所有应用共用的基础配置（如应用发布、语义精简协议）
- **极速应用类配置**：面向 AIUI 最新的极速超拟人应用，提供全新的基础配置、语义模型、回复角色、全双工、表情、长时记忆、声纹等能力
- **传统应用类配置**：面向传统语义与通用大模型语义链路的老应用，提供 ASR、结构化语义、星火大模型、全链路合成等能力

## 使用流程

1. 登录 [AIUI 平台](https://aiui.xfyun.cn/) →「我的应用」
2. 选择目标应用 →「应用配置」
3. 根据应用类型选择对应分类下的配置项进行设置
4. 保存后生效，部分能力需发布应用后才会作用于线上交互

::: tip 提示
极速应用与传统应用的配置项互不通用，请先确认你的应用类型，再选择对应分类下的配置。
:::

## 极速应用类配置

基于 AIUI 平台最新的极速超拟人应用做配置说明。

<div class="nav-cards">

- **[基础配置](./basic-config)** — 支持识别引擎选型、兜底回复大模型选型、回复角色选型。
- **[语义模型配置](./semantic-model)** — 极速交互链路下技能和大模型能力融合配置项，支持传统语义技能、新智能体、文档问答等能力配置。
- **[回复角色配置](./reply-role)** — 介绍全链路合成播报发音人配置。
- **[语音识别配置](./asr-config)** — 介绍识别引擎效果参数设置。
- **[应用后处理配置](./post-process)** — 支持配置客户自有服务地址，接受 AIUI 交互中服务数据转发云云对接。
- **[三方大模型配置](./third-party-llm)** — 配置第三方大模型接入参数，包括 API 密钥、请求超时设置及模型选择。
- **[流畅全双工交互配置](./full-duplex)** — 配置流畅全双工交互，三层 VAD 能力：语音 VAD、动态判定、智能聆听。
- **[表情标签配置](./emotion-tags)** — 配置表情标签，在聊天气泡旁展示对应的 Emoji，根据情绪改变交互界面设计。
- **[长时记忆配置](./long-memory)** — 配置长时记忆，基于历史数据构建用户画像，提供千人千面的对话服务。
- **[声纹识别配置](./voiceprint)** — 配置声纹识别，同一设备可根据声纹单独存储长时记忆。

</div>
