::: info 概述
AIUI 提供多种配套硬件模组，覆盖降噪板、评估板、多模态套件、离线语音模块和 USB 声卡等产品形态。每款硬件均附带产品白皮书、使用手册和开发指南，方便开发者快速完成硬件集成与调试。
:::
