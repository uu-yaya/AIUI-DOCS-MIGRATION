# 接口连接指南

**第一步**，将麦克风阵列连接至RK3328降噪板。

麦克风阵列连线采用2X8-2.0牛角排线，将插头按照缺口方向插入插座。

![连接指南图片1](/media/202308/1_1691028435.jpg "null")
图2 RK3328麦克风阵列接线图示

**第二步**，将扬声器连接到RK3328降噪板。

扬声器接口采用PH2.0-4P座，将插头按照缺口方向插入插座。如更换扬声器，请参照接口线序标识，L-代表左声道负极，L+代表左声道正极，R-代表右声道负极，R+代表右声道正极。如果采用外置功放，此接口不接。

![连接指南图片2](/media/202305/2023-05-03_152952_6069880.8509316240609863.png "null")
图3 RK3328扬声器接线图示

**第三步**，将功放/回采信号输入连接至RK3328降噪板。

功放/回采信号接口采用3.5（三段式插头）耳机插孔或PH2.0-3P插座。可采用一根3.5音频线，与上位机音频输出接口连接。
![连接指南图片3](/media/202305/2023-05-03_153007_2317760.7860976209244719.png "null")
图4 RK3328功放/回采接线图示

**第四步**，将麦克风音频输出接口连接至上位机。

接口输出的信号，是由RK3328降噪板处理后的麦克风信号，如回声消除、指向识别、环境降噪等。该接口采用3.5（三段式插头）耳机插孔或PH2.0-3P插座。可采用一根3.5音频线，与上位机麦克风接口连接。
![连接指南图片4](/media/202305/2023-05-03_153018_8959390.8352822843178719.png "null")
图5 RK3328麦克风音频输出接线图示

**第五步**，将TTL串口连接到上位机。

TTL串口采用PH2.0-3P插座，将插头按照缺口方向插入插座。为3.3VTTL电平，可采用USB转TTL与上位机连接。请参照RK3328降噪板的TTL串口接口线序标识，TX为数据发送，RX为数据接收，GND为供地。

![连接指南图片5](/media/202305/2023-05-03_153028_8639280.13210583883845628.png "null")

图6 RK3328TTL串口接线图示

**第六步**，将电源连接到RK3328降噪板。

电源接口采用DC5.5-2.1插座或PH2.0-2P插座，将插头按照缺口方向插入插座。电源电压为DC12V，使用板载功放时，提供电流不低于3A。不使用板载功放时，提供电流不低于1A，请参照RK3328降噪板的电源接口线序标识，DC12V为电源正，GND为电源负。
![连接指南图片6](/media/202305/2023-05-03_153041_2537210.5585419621544723.png "null")
图7 RK3328电源线连接图示

# 使用方法

本章介绍两种使用方式，如果自带功放满足扬声器声音大小需求，则使用板载功放。若板载功放无法满足需求，则可以使用外接功放。

## 使用板载功放

使用RK3328降噪板板载功放时，上位机音频输出信号直接连接至RK3328降噪板。并将2个≤10W4Ω喇叭连接到RK3328降噪板扬声器接口上。
![使用方法1](/media/202305/2023-05-03_153117_5692920.6795102595511936.png "null")

图8 RK3328降噪板板载功放接线图示

![板载功放图片2](/media/202303//1679563212.7359855.png "null")

图9 采用板载功放接线图示

## 使用外置功放

不使用RK3328降噪板板载功放时，上位机音频输出信号经过1分2音频线，将音频信号分别给外置功放和RK3328降噪板功放/回采音频输入接口。此时RK3328降噪板功放/回采音频输入接口只起到回采信号反馈功能。且RK3328降噪板扬声器接口，将不再需要接扬声器。

# 快速体验指南

使用PC机进行快速体验，可按照下列示意图接线。

![1678350613998(1)](/media/202303//1679563212.7698774.jpeg "null")

图12 实物接口连接示意图

**第一步**，将所有连接按照示意图连接好，在PC端打开讯飞串口工具：（[【附件】3328串口调试工具\_win32.zip](/media/attachment/2024/12/3328%E4%B8%B2%E5%8F%A3%E8%B0%83%E8%AF%95%E5%B7%A5%E5%85%B7_win32.zip)）。
如需在Linux端实现串口，可以下载C语言示例代码:
[【附件】串口实现参考\_C语言.zip](/media/attachment/2024/12/%E4%B8%B2%E5%8F%A3%E5%AE%9E%E7%8E%B0%E5%8F%82%E8%80%83_C%E8%AF%AD%E8%A8%80.zip)

![](/media/202412/2024-12-26_135711_2571940.9072215695752557.png "null")

图13 串口工具界面

**第二步**，查看板卡加载串口的端口号：
可以从设备管理器查看3328降噪板连接串口后对应的端口号（当有多个端口时，可以通过重新拔插刷新端口信息来确认）。选择识别出来的USB转TTL串口号，如果USB转TTL没有驱动，可以下载附件的驱动进行安装（[【附件】CH340SerSetup.zip](/media/attachment/2024/12/CH340SerSetup.zip)）。

![](/media/202412/2024-12-26_140002_0659570.17150350972349326.png "null")

**第三步**，配置正确的端口号后点击”打开串口”。
提示“串口通信正常”代表串口连接成功了。

![](/media/202412/2024-12-26_135753_5554750.5531381311272224.png "null")

**第四步**，体验语音唤醒唤醒、声源定位、噪声抑制和回声消除。

对着麦克风说“小飞小飞”，这时电路板被唤醒，串口将输出唤醒信息，信息中包含使用的唤醒词信息、唤醒角度信息等。

打开PC机录音软件开始录音，此时可以从刚才唤醒的方向对着麦克风说话，也可以从其他的角度对着麦克风说话,也可以在PC机上播放音乐，在进行从不同角度对着麦克风说话（注意喇叭不要离麦克风太近，以免喇叭声音过高造成失真严重）。结束录音，收听录音内容。正常情况下，可以听到唤醒方向的说话声音洪亮清晰，其他角度声音小且模糊。PC机播放的音乐，会因为回声消除算法在录音中基本听不到。

# 替换唤醒词资源

RK3328降噪板内置唤醒词资源版本为:3.17.7引擎版本，开发者可以根据需要选择下面两种方式进行唤醒词资源更换。

## 5.1. 发送串口消息更换唤醒词

通过串口工具直接发送唤醒词替换命令（[详见协议手册 4.1.1 唤醒词更换说明](/hardware/rk3328-nr/protocol "详见协议手册 4.1.1 唤醒词更换 说明")）即可快速更换唤醒词资源，下面以串口工具操作做演示：
1、降噪板连接到PC端
2、在PC端打开讯飞串口工具，配置端口号，点击打开串口
![](/media/202411/2024-11-21_164708_0383250.635435081034065.png "null")
3、在串口工具左下角输入框输入需要更换的唤醒词命令，点击 “Send Raw”不报错即可：
![](/media/202411/2024-11-21_165409_1661460.5822409910175347.png "null")

温馨提示

1、keyword 取值为唤醒词拼音，注意多个唤醒词间用" ; "隔开

2、threshold 取值为唤醒词阈值，注意多个唤醒词间用" ; "隔开

```json
{
     "type": "wakeup_keywords",
     "content": {
         "keyword": "xiao3 fei1 xiao3 fei1;xiao3 bu4 xiao3 bu4",
         "threshold": "900;900"
     }
}
```

4、语音唤醒验证修改结果
![](/media/202411/2024-11-21_165456_7152700.5240355517309898.png "null")

## 5.2. 本地唤醒词资源替换

当前3328降噪板串口修改的唤醒词属于浅定制 中文资源，当有深定制、或其他资源线下获取时，也可以通过 adb 命令进行替换使用。演示流程如下：
1、登陆AIUI平台进行AIUI应用,在”接入配置”页面,下载 3.17.7引擎版本 唤醒词资源
![](/media/202411/2024-11-21_165819_1750680.0023936471164259787.png "null")
2、解压下载的资源压缩包，获取唤醒词资源 res.bin 文件
3、降噪板连接到PC端，使用 adb push命令导入唤醒词资源，然后进行重启即可

```bash
adb push res.bin  /oem/build/vtn/res.bin
adb reboot
```
