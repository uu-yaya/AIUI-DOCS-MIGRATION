## 功能描述

| 功能 | 说明 |
| --- | --- |
| 远场5m拾音 |  |
| 声源定位 | 支持 180°（线性4麦和线性6麦）和360°（环形6麦）方向定位。安静场景精度±15° |
| 回声消除 | 在播放和录音同时进行的交互场景，将扬声器的声音屏蔽，只接收用户的声音。 |
| 定向拾音 | 用波束形成技术，对波束内声音增强，波束外声音进行抑制。具体见《RK3328降噪板协议手册.docx》的3.2节波束。 |
| 功放输出 | 模块自带双路 3W 功放芯片。可直接接入喇叭进行音频播放。 |
| 语音唤醒 | 用户通过指定唤醒词的语音信号可以唤醒模块，也可设置模块开机自动唤醒。模块唤醒会上报唤醒信息，包括唤醒角度、唤醒得分、所属波束、唤醒词等。 |
| 音频降噪 | 首次唤醒后，持续输出降噪后音频。 |
