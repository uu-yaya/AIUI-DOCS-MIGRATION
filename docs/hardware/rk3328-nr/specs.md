# 产品概述

RK3328降噪板是一款基于多麦克风阵列的语音前端解决方案。模块采用四核Cortex-A53 的 CPU，频率最高 1.5GHz。内部集成科大讯飞语音算法，利用麦克风阵列的空域滤波特性，通过唤醒人的角度定位，形成定向拾音波束，并对波束以外的噪声进行抑制，提升远场拾音质量。同时针对人机交互一体终端，集成高性能回声消除算法，降低语音、语义识别难度，是客户平台集成的首选方案。
