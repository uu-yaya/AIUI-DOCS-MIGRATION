## 产品概述

RK3588 AIUI多模态交互产品是一款基于多麦克风阵列+图像识别的整体语音解决方案。芯片采用的是瑞芯微RK3588，8nm生产工艺的8核CPU（四核Cortex-A76 的 CPU，频率最高 2.4GHz ；四核Cortex-A55的CPU，频率最高 1.8GHz）。接口方面集成千兆以太网、HDMI、HDMI IN、RS232 串口、RS485、USB3.0、USB2.0、TypeC(DP)。本产品集成神经网络处理器 NPU，算力高达6.0Tops，兼容多种 AI框架，可灵活应用于各种智能产品。

## 适用范围

RK3588 AIUI多模态交互产品方案广泛适用于集群服务器、高性能计算/存储、计算机视觉、边缘计算、商显一体设备、医疗健康设备、自动售货机、工业设备等 AI 应用领域。

## 产品特点

- **高性能四+四大小核处理器**

采用RK3588系列，标准6Tops 算力

- **产品长时间稳定运行，具有高可靠性**

大风扇散热，高效降温。60℃高温老化 7x24 小时稳定运行

- **工业级外观的外观设计**

外表美观大气、多种安装方式、隐蔽式引线孔

- **多种网络连接方式**

支持千兆以太网（RJ45）

- **串口通讯连接**

支持 RS232\*2

- **操作系统**

操作系统：Linux/Android，支持 Android 系统定制，提供系统调用接 API 参考代码

- **配置4-8麦克风阵列板**

可配置近场麦克风阵列板，支持 4-8 个模拟麦、支持关键词唤醒、回声消噪、VAD（端点检测）(VoiceActivation Detection)和声源定位等功能，大大提升了语音识别的精度与准确性。
