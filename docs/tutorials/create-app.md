## 前置条件

- 已注册科大讯飞账号并完成实名认证
- 已登录 [AIUI 开放平台](https://aiui.xfyun.cn)

## 你将完成的目标

通过本教程，你将学会：

1. 在 AIUI 平台创建一个新应用
2. 配置语义模型、语音识别和语音合成
3. 提交审核并发布应用上线
4. 获取 AppID、APIKey、APISecret 等接入凭证

## 第一步：创建应用

登录 AIUI 账号，进入 [我的应用](https://aiui.xfyun.cn/app) 页面，点击 **创建应用** 按钮。

填写应用基本信息：

- **应用名称**：为你的应用取一个有意义的名称
- **应用描述**：简要说明应用的用途和场景

::: tip 应用类型说明
2025-06-12 之后创建的应用均为 **极速交互** 类型，绑定极速超拟人交互方案，语义和大模型配置在服务链路中固定开启。如需了解不同应用类型的区别，请参阅 [AIUI 应用介绍](/platform-service/app-intro)。
:::

## 第二步：配置语义模型

进入应用配置页面，完成语义模型设置：

1. 在 **模型选择** 区域，选择「讯飞星火交互大模型 V2」
2. 进入 **结构化语义配置**，点击「添加」
3. 开启所需的技能并保存
4. 在页面右侧的模拟测试区域验证配置效果

## 第三步：配置语音合成

AIUI 提供两种合成方式：

| 合成方式 | 说明 | 适用场景 |
|---------|------|---------|
| 主动合成（推荐） | 开发者主动调用合成接口 | 需要精细控制合成时机 |
| 语义后合成 | 语义理解后自动合成语音 | 所有技能回复自动播报 |

在 **语音合成配置** 区域选择合适的发音人，可试听效果后确认。

## 第四步：配置语音识别

1. 在应用配置中，将 **识别引擎** 设为「通用-中文-近场」
2. 展开语音识别配置，根据需要勾选「识别结果优先阿拉伯数字」
3. （可选）上传热词文件以提升特定词汇的识别率：
   - 点击「下载热词模板」
   - 按每行一个词语的格式填写
   - 上传至平台，约 20 分钟后生效

## 第五步：提交审核

配置完成后，点击 **审核上线**，一般 24 小时内处理完毕。

## 第六步：发布应用

审核通过后，页面上会出现「发布」和「版本管理」按钮：

1. 点击 **发布**，按要求填写发布信息
2. 发布后，测试情景模式（`main_box`）的配置将同步到线上环境（`main`）

::: warning 注意
发布操作会将当前测试配置覆盖到线上环境，请确认配置无误后再发布。
:::

## 获取接入凭证

应用创建后，在应用信息页面可查看以下凭证：

- **AppID**：应用唯一标识
- **APIKey**：接口调用密钥
- **APISecret**：接口调用密钥（用于签名鉴权）

这些凭证将在 SDK 或 API 接入时使用，请妥善保管。
