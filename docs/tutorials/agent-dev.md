## 前置条件

- 已创建 AIUI 应用（参考 [创建应用教程](/tutorials/create-app)）
- 已登录 [AIUI 技能工作室](https://aiui.xfyun.cn/studio/agent)
- 了解意图识别的基本概念

## 你将完成的目标

通过本教程，你将学会：

1. 理解智能体与技能的区别
2. 创建本地智能体并定义意图
3. 编写和泛化语料
4. 在应用中配置和使用智能体

## 什么是 AIUI 智能体

AIUI 智能体（Agent）是基于大语言模型，能够执行复杂任务的应用程序。智能体根据用户的语音请求，智能判断交互意图并进行任务分发。

### 智能体 vs 技能

| 对比项 | 智能体 | 技能 |
|--------|--------|------|
| 底层能力 | 基于大模型 | 基于传统 NLP |
| 意图理解 | 大模型智能判断 | 语料匹配 |
| 语料泛化 | 大模型自动泛化 | 需人工编写 |
| 槽位抽取 | 不支持 | 支持 |
| 适用场景 | 设备命控、任务分发 | 结构化信息提取 |

### 智能体类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 官方智能体 | AIUI 平台针对通用场景打造 | 美食顾问、旅行助手 |
| 本地智能体 | 调用端侧应用执行任务 | 投影仪命控、家电控制 |

## 第一步：创建智能体

1. 进入技能工作室的 [智能体](https://aiui.xfyun.cn/studio/agent) 页面
2. 点击 **创建智能体**
3. 填写智能体名称和描述

::: tip 当前限制
目前仅支持创建「本地智能体」类型。官方智能体需联系商务/技术支持获取。
:::

## 第二步：创建意图

进入智能体后，创建需要识别的意图。

以「投影仪命控」智能体为例，可创建以下意图：

| 意图名称 | 意图标识 | 用户说法示例 |
|---------|---------|------------|
| 对焦 | `focus` | "画面虚焦了怎么调整" |
| 调节音量 | `volume` | "声音太小了" |
| 切换信源 | `switch_source` | "切到 HDMI 输入" |

::: warning 与技能的区别
智能体只提供意图落域能力，不提供填槽抽取能力。如果需要从用户话语中提取参数，请使用自定义技能。
:::

## 第三步：编写语料

为每个意图编写用户常见的说法（语料），覆盖多种表达方式：

```text
意图：对焦

语料：
- 这个投影仪画面虚焦了，怎么调整呢
- 画面不清楚，帮我调一下
- 对焦一下
- 图像模糊了
```

## 第四步：泛化语料

智能体支持大模型自动泛化功能，可基于已有语料生成更多相似表达：

1. 编写好基础语料后，点击 **泛化** 按钮
2. 大模型会自动生成类似的语料
3. 可以点击「一键添加」添加全部泛化语料
4. 也可以逐条筛选，只添加合适的语料

::: tip 泛化建议
- 先编写 5-10 条高质量基础语料
- 泛化后审核结果，删除不合适的语料
- 多轮泛化可以覆盖更多表达方式
:::

## 第五步：构建和测试

1. 点击 **构建**，等待智能体构建完成
2. 在测试窗口中输入文本验证意图识别效果
3. 确认各意图识别准确后，点击 **发布**

## 第六步：在应用中配置

1. 进入 AIUI 应用配置页面
2. 在 **语义模型配置** 中添加智能体
3. 保存并发布应用

### 端侧对接

智能体返回的结果包含识别到的意图信息，端侧应用需要根据意图执行对应的操作：

```text
用户：这个投影仪画面虚焦了，怎么调整呢
→ 智能体识别意图：focus
→ 端侧执行对焦操作
```

具体的结果解析格式请参考 [交互结果协议说明](/sdk-dev/basics/result-protocol)。
