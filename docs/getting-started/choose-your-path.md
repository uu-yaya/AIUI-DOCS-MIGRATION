## 概述

AIUI 提供三种服务链路和两种集成方式（SDK / WebSocket API）。本文帮助你根据项目需求快速做出选择。

## 三种服务链路对比

| 维度 | 传统语义交互链路 | 通用大模型交互链路 | 极速超拟人交互链路 |
|---|---|---|---|
| 推出时间 | 2016 年底 | 2022 年 | 2025 年 6 月 |
| 语义核心 | 语法句式匹配（结构化语义） | 大模型级联（意图落域、知识分类、智能体等） | 大模型级联升级版 |
| 连接方式 | 短连接（会话级） | 长连接（多轮对话） | 长连接（多轮对话） |
| SDK 版本要求 | 5.x 直接对接；6.x 需 `aiui_ver=1` | 6.x 及以上，`aiui_ver=2` | 6.8.x 及以上，`aiui_ver=3` |
| 大模型支持 | 不支持 | 支持（星火大模型） | 支持（星火大模型 + 三方大模型） |
| 智能体 | 不支持 | 支持 | 支持（含三方平台接入、编排和结果输出） |
| 三方大模型转发 | 不支持 | 不支持 | 支持（兼容 OpenAI API 协议） |
| 应用类型 | 老应用 | 老应用（需开启大模型开关） | 新应用（2025-06-12 后创建） |

### 传统语义交互链路

以结构化语义理解为核心，采用语法句式匹配方案。支持的能力项包括：商店技能、自定义技能、语句问答、关键词问答、设备人设、闲聊兜底。

**服务流程**：语音识别 → 结构化语义 → 后处理 → 语音合成

**技能优先级**：自定义技能 > 语句问答 > 设备人设 > 商店技能 > 关键词问答 > 兜底闲聊

::: warning 注意
传统语义链路**不可开启大模型配置**。已有传统语义链路授权的老应用，大模型能力将不再支持随意开关。
:::

### 通用大模型交互链路

在传统语义基础上，通过级联方式组合语义规整、意图落域、知识分类、智能体、文档问答、联网搜索、兜底大模型等多种能力。

**适用场景**：需要更灵活的自然语言理解、多轮对话、知识问答等场景。

### 极速超拟人交互链路

通用大模型链路的升级版，新增以下能力：

- 智能体支持三方平台接入、编排和结果输出
- 兜底大模型支持开发者自行选配
- 支持兼容 OpenAI API 协议的三方大模型配置转发
- 支持全双工流畅交互、表情标签、长时记忆等高级特性

::: tip 推荐
**新项目建议选择极速超拟人交互链路**。该链路绑定新应用，功能最完整，且持续更新。
:::

## SDK 与 API 的选择

| 维度 | SDK 集成 | WebSocket API 集成 |
|---|---|---|
| 适用平台 | Android、iOS、Windows、Linux | 任意支持 WebSocket 的平台 |
| 离线能力 | 支持（唤醒、离线识别、离线合成等） | 不支持 |
| 前端声学 | 支持（麦克风阵列、回声消除等） | 不支持（需自行处理音频） |
| 集成复杂度 | 导入 SDK 库，配置 `aiui.cfg` | 按协议发送/接收 WebSocket 消息 |
| 典型场景 | 智能音箱、机器人、嵌入式设备 | Web 应用、服务端对接、跨平台轻量集成 |

## 应用版本说明

### 新应用（极速交互）

- **创建时间**：2025-06-12 之后
- **服务链路**：固定为极速超拟人交互链路
- **免费额度**：10,000 次/年、API 并发 3 路、AIUI 装机量 20 台、唤醒装机量 20 台

### 老应用

**支持传统语义 + 通用大模型**（2022-06-01 至 2025-06-11 创建）：

- 可通过大模型开关切换服务链路
- 应用信息包含 AppID、APIKey、APISecret
- 免费日额度：500 次（传统语义）/ 500 次（大模型），API 并发 20 QPS

**仅支持传统语义**（2022-06-01 之前创建）：

- 不支持大模型配置
- 应用信息仅包含 AppID 和 APIKey，无 APISecret

## 快速决策流程

1. **新项目？** → 创建新应用，使用[极速超拟人链路](./hello-world-rapid)
2. **已有老应用，想用大模型？** → 开启大模型开关，使用[通用大模型链路](./hello-world-llm)
3. **已有老应用，只需结构化语义？** → 继续使用[传统语义链路](./hello-world-traditional)
4. **设备端需要离线能力？** → 选择 SDK 集成
5. **Web 端或服务端对接？** → 选择 WebSocket API 集成
