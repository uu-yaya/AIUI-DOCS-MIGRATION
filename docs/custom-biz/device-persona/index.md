::: info 概述
设备人设是 AIUI 为智能硬件提供的拟人化能力，通过配置年龄、性别、姓名、爱好等 46 种属性及对应回复语，让设备具备独特的"人设"，使产品交互更加自然、富有温度。
:::

## 什么是设备人设

设备人设（Device Persona）让每一台设备都有自己的"性格档案"。当用户询问设备相关的个性化问题时（如"你叫什么名字"、"你多大了"），设备会基于人设配置给出符合角色定位的回复，而不是千篇一律的机械应答。

AIUI 平台内置默认人设**蓝小飞**，开发者可以：

- 在蓝小飞基础上编辑属性值和回复语
- 新建自定义人设，为不同产品线打造差异化的虚拟形象
- 配置基础信息、联系方式、社会关系等多维度属性

## 核心能力

- **46 种属性**：覆盖姓名、年龄、性别、生日、爱好、家庭关系、联系方式等
- **系统回复 + 自定义回复**：AIUI 提供默认回复池，开发者可补充自定义文本回复
- **智能关联**：如配置生日后自动推导年龄、星座、生肖
- **多应用场景**：广泛用于智能音箱、机器人、儿童玩具等硬件产品

## 使用流程

1. 登录 [AIUI 平台](https://aiui.xfyun.cn/) →「我的应用」
2. 选择目标应用 →「应用配置」→「语义理解」
3. 在「管理技能配置」中开启「设备人设」
4. 跳转至 [技能工作室 · 设备人设](https://aiui.xfyun.cn/studio/character) 配置属性值与回复语

**优先级说明**：自定义技能 > 自定义问答 > 设备人设 > 商店技能
