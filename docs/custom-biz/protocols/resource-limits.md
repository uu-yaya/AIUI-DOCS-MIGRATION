---
title: 技能资源限制
---

::: info 概述
已下关于AIUI技能工作室技能或问答库创建资源限制说明,针对的都是**单个AIUI平台账号**，而不是**某个AIUI应用**。
:::

## 自定义技能限制

| 项目 | 限制 |
| --- | --- |
| 技能个数 | 20个 |
| 一个技能下的意图数 | 40个 |
| 一个意图下的提问语料数 | 500个 |
| 一个技能下的槽位数 | 60个 |
| 一个意图下的槽位数 | 40个 |
| 技能导入/导出次数 | 导入：10次/天 |

## 问答库限制

| 项目 | 限制 |
| --- | --- |
| 问答库个数 | 40个 |
| 一个问答对问题数 | 40个 |
| 一个问答对答案数 | 20个 |
| 一个问答库下的问答数 | 20000个问答对或50000个问题 |
| 问答库导入的文件大小/导入次数 | 文件大小：<5M  导入次数：30次/天 |

## 文档问答限制

| 项目 | 限制 |
| --- | --- |
| 文档问答个数 | 10个 |
| 一个文档问答支持上传的文件个数 | 10个 |
| 单个文件大小 | < 30M |
| 单个文件最大可拆分知识点数 | 3400个 |

## 实体限制

| 项目 | 限制 |
| --- | --- |
| 实体个数 | 40个 |
| 一个词条类实体下的词条数 | 20000个 |
| 实体导入的文件大小/导入次数 | <5M 30次/天 |
| 一个动态实体下的资源数 | 5个 |

签约客户可联系[技术支持](https://aiui.xfyun.cn/ask/submit)，免费提额。
