::: info 概述
技能开发流程涵盖从创建技能、配置意图与语料、到测试验证和审核发布的完整链路。本节还提供了技能后处理开发和批量导入导出等进阶功能说明。
:::

## 本节内容

<div class="nav-cards">

- **[创建技能和意图](/custom-biz/skill-studio/development/create)** — 技能标识，技能类型不可修改。
- **[技能导入导出](/custom-biz/skill-studio/development/import-export)** — 对某一个技能进行批量导出操作。你可以导出后进行本地修改，修改完成后导入进行覆盖，也可以导出后再导入另一个技能，达到技能复制的目的。
- **[意图配置](/custom-biz/skill-studio/development/intent-config)** — 意图配置主要就是语料填充编写，关于语料编写详细流程如下：
- **[技能后处理](/custom-biz/skill-studio/development/post-process)** — 技能后处理帮助开发者实现自定义功能，可直接与万维网通信。
- **[技能发布](/custom-biz/skill-studio/development/publish)** — 技能测试和发布前，都需要先进行技能构建。
- **[技能测试](/custom-biz/skill-studio/development/testing)** — 技能和实体的配置保存并构建完成后，在页面右侧的测试窗，输入“查询顺丰的12345”，可查看结构化数据结果。

</div>
