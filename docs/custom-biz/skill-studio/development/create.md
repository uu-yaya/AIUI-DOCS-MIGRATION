---
title: 创建技能和意图
---

::: info 概述
本文介绍如何在技能工作室中创建自定义技能和新建意图，是技能开发的第一步。
:::

## 创建技能

进入 AIUI 技能工作室，点击「创建技能」按钮，填写技能名称、技能标识和技能类型等基本信息。

![创建技能](/images/custom-biz/create-skill.png)

::: warning 注意
技能标识和技能类型创建后不可修改，请谨慎填写。
:::

## 新建意图

技能创建完成后，在技能详情页面点击「新建意图」，为技能添加用户交互意图。每个意图代表一类用户请求，需要配置意图名称和意图标识。

![新建意图](/images/custom-biz/create-intent.png)
