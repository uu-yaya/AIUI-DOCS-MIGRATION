## 概述

AIUI 平台支持配置三方大模型作为回复大模型。

::: warning 前提条件
配置的三方大模型服务需要满足 OpenAI API 协议。
:::

配置需要在开源的三方平台获取以下信息：

| 配置项 | 说明 |
| --- | --- |
| 接口地址 | API 接入的 URL |
| 鉴权信息 | 应用的 API Key |
| 模型标识 | 模型的 Domain 标识 |

以下以通义千问为例介绍配置步骤。

## 步骤一：获取三方平台信息

1. **注册**：前往阿里云注册账号并完成实名认证。


2. **开通模型服务**：在阿里云百炼页面点击"立即开通"。


3. **获取 API Key**：在百炼控制台创建 API Key 并复制。


4. **获取接口地址**：在官方 API 接入文档中获取接入 URL，例如：

   ```text
   https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions
   ```


## 步骤二：创建 AIUI 应用

在 AIUI 平台点击"我的应用"，创建新应用。


## 步骤三：模型配置

1. **进入配置界面**：在应用配置页选择"模型 > 三方大模型"，点击"模型配置"。


2. **填写配置信息**：

   | 字段 | 填写内容 |
   | --- | --- |
   | 接口地址 | 步骤一获取的 URL |
   | 鉴权信息 | 步骤一获取的 API Key |
   | 模型标识 | 在三方模型广场查找的模型 Domain（如 `qwen-plus`） |


3. **测试**：输入问题，若有回复则配置成功。
