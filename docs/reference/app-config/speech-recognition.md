## 概述

语音识别配置是 AIUI 应用中对链路使用的语音识别引擎能力进行控制的配置项。

## 新老应用区分

由于 AIUI 应用类型不同，新极速交互类应用和老应用有细微差异：

| 应用类型 | 识别引擎配置位置 |
| --- | --- |
| 新极速类应用 | 识别引擎选择在**基础配置**项下 |
| 老应用 | 识别引擎选择在本配置项下 |



## 识别引擎选择

在老应用中，语音识别配置项支持识别引擎选择，常见分类方式：

| 分类维度 | 可选值 |
| --- | --- |
| 声学模型 | 中文近场、中文远场 |
| 语言 | 中文、英文 |
| 中文方言 | 中文普通话、四川话、粤语、东北话等 |
| 交互场景 | 餐饮-中文、车载-中文、智能家居-中文等 |

::: info 说明
交互场景类引擎在对应领域定向优化了行业名词，可提高该场景下语音交互的识别率。
:::


## 识别设置

该配置项主要设置识别引擎结果输出相关类型，不区分应用类型：

| 配置项 | 说明 |
| --- | --- |
| 识别添加标点 | 识别结果最后添加标点 |
| 识别结果优先阿拉伯数字 | 示例：识别为 `1岁` 而非 `一岁` |
| Progressive 流式识别 | 简称 PGS，说话过程中返回多次中间结果并不断修正 |


在老应用中还支持以下特性配置：

| 配置项 | 说明 |
| --- | --- |
| 所见即可说 | 一次性热词，提高识别率 |

## 识别热词

当语音交互中某些名词的识别结果不符合预期时，可配置识别热词提高识别率。

- **上传识别热词** — 永久热词，提高识别率。


## 识别敏感词过滤

该配置项用于对识别结果做敏感词拦截，支持自定义设置和白名单，具体编辑方式可下载模板了解。

- **识别敏感词过滤** — 将敏感词过滤为 `*`，不展示给用户。默认关闭，开启后需解析。
