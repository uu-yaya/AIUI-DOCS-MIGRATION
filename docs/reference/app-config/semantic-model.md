## 概述

语义模型配置是 AIUI 极速超拟人交互类应用提供的配置项目，主要为开发者提供传统语义技能（官方技能、问答库）、大模型语义功能模块（文档问答库、智能体、联网搜索）相关能力配置选项。

平台分类如下：

- **添加工具** — 配置问答、智能体、技能等能力
- **搜索设置** — 配置联网搜索和知识溯源

## 添加工具

### 问答

该配置项用于为极速链路配置自定义大模型链路的**文档问答**。

点击"添加"，可配置当前账号下已创建并发布的文档问答。开发者还可以配置文档问答在服务链路的集成策略（针对所有文档问答生效）：

| 配置项 | 可选值 | 说明 |
| --- | --- | --- |
| 检索方法 | 单向量召回 / 多路召回 | 计算知识片段与用户问题相关性的方法 |
| 相关性阈值 | 0 ~ 1 | 低于阈值的知识片段将被排除 |

::: info 知识说明
- **相关性**：知识片段和用户问题的匹配程度，数值在 0 到 1 之间，值越高匹配度越好。
- **检索方法**：单向量召回依靠语义向量相似度计算；多路召回将语义向量、关键词等多种方式的相似度合并计算。
- **相关性阈值**：例如阈值设为 0.6，则只保留相关性 ≥ 0.6 的知识片段，避免不相关知识干扰回答。
:::


### 智能体

该配置项用于为极速链路配置自定义大模型链路的**新版智能体**。

点击"添加"，可配置当前账号下已创建并发布的新版智能体；点击"创建智能体"可跳转到智能体开发界面。

::: info 知识说明
- **旧版智能体**：仅提供意图匹配能力，开发者编辑时仅提供语料设置。
- **新版智能体**：除意图匹配能力外，开发者可按业务需要进行节点能力模块自定义编辑，与目前主流智能体开发方式基本一致。
:::


### 技能

该配置项用于为极速链路配置传统语义技能能力，支持类型：

- 官方技能
- 自定义技能
- 语句问答
- 关键词问答

还支持特性设置：自定义技能或自定义语句问答的阈值设置、关键词过滤、语义敏感词、精简协议。

#### 技能配置

点击"添加"，可在当前应用的情景模式下配置需要的技能。


#### 自定义技能阈值设置

在自定义技能添加界面，勾选自定义技能后，点击技能下的设置图标进行设置。

::: warning 注意
仅支持 **自定义私有技能** 和 **自定义语句问答** 两种技能类型。
:::

自定义阈值用于调节技能匹配度，可根据模糊匹配度和误识别两个维度综合考虑设置。AIUI 技能实际触发的模糊匹配程度可查看技能 JSON 中的 `source` 字段取值。

| 阈值方向 | 效果 |
| --- | --- |
| 阈值越高 | 语句格式要求越严，模糊匹配越低，不容易误识别 |
| 阈值越低 | 语句格式要求越松，模糊匹配越高，容易误识别 |


#### 技能设置

技能配置后，可点击"设置"进行高级能力配置：

- **关键词过滤** — 过滤识别结果中的关键词，不进行语义理解。常用于过滤唤醒词。
  - 过滤前：`小飞小飞，你好`
  - 过滤后：`你好`

- **语义敏感词过滤** — 过滤用户回复信息中的敏感词，支持自定义增加。触发时固定响应：

  ```json
  {
    "answer": {
      "text": "文明社会，请文明用语。",
      "topic": "",
      "type": "T"
    },
    "operation": "ANSWER",
    "rc": 0,
    "service": "forbidden",
    "sid": "xxxxxxxxx",
    "state": null,
    "text": "xxx",
    "uuid": "xxxxxxxxx"
  }
  ```

  ::: info 说明
  极速交互链路在服务层面各个模组的输入输出都有单独的安全审核服务进行监控。
  :::

- **精简协议** — 简化结构化语义，将技能结果分为媒资类和播报类。默认关闭。将多个技能精简成一种输出协议，大大降低技能适配工作，适用于无屏设备。


## 搜索设置

### 搜索引擎

提供大模型联网搜索能力项，默认关闭。

开启后，用户进行语音交互时，当服务判断需要进行搜索，会调用联网服务进行信息搜索，然后由大模型对搜索信息进行总结答复。


### 知识溯源

该能力项控制大模型服务结果的下发。通常作为服务链路中加载文档问答或联网搜索获取的知识点，内容较多，默认不下发。

开启后，AIUI 服务在交互时会新增下发 `cbm_knowledge` 结果，开发者可据此了解大模型回复内容的来源。
