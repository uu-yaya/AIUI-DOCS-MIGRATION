## 概述

基础配置是 AIUI 极速超拟人交互类应用提供的配置项目，为开发者提供快速模块能力选型配置，包括：

- **识别引擎** — 语音交互时音频转文字能力的服务配置
- **兜底大模型** — AIUI 服务链路中最后的兜底回复模型选型
- **回复角色** — 大模型设备人设方案配置

## 识别引擎

识别引擎主要针对语音交互时音频转文字能力提供的服务配置项，开发者按需选择配置即可。


::: tip 温馨提示
由于极速链路中针对识别引擎需要适配语义 VAD（端点检测）能力，如有非列表下其他识别引擎需求，可反馈 AIUI 技术同事评估。
:::

## 兜底大模型

兜底大模型是 AIUI 服务链路中最后的兜底回复模型，目前支持以下选项：

| 模型名称 | 说明 |
| --- | --- |
| 讯飞星火交互大模型 V2 | AIUI 官方模型，授权包含在极速应用授权中 |
| 讯飞星火儿童交互大模型 | AIUI 官方模型，授权包含在极速应用授权中 |
| 讯飞星火 Spark 4.0 Ultra | AIUI 官方模型，授权包含在极速应用授权中 |
| DeepSeek-V3 | AIUI 官方模型，授权包含在极速应用授权中 |
| 三方大模型 | 需满足 OpenAI API 协议，开发者自行管理授权 |


AIUI 官方提供的模型选配（交互大模型、儿童大模型、星火 4.0 Ultra、DeepSeek-V3）调用授权（Token 和并发）都包含在 AIUI 极速应用授权中，开发者可以忽略。

三方大模型配置需选择满足 OpenAI API 协议的大模型进行配置，包含以下三个信息：

- **接口地址**
- **鉴权信息**
- **模型标识**

三方大模型配置后，AIUI 仅做托管调用，涉及的 Token 或并发授权需要开发者侧自行处理。详见[三方大模型配置](/reference/app-config/third-party-llm)文档。


## 回复角色

回复角色是 AIUI 平台提供的大模型设备人设类方案，除了包含官方设置好的角色以外，也支持开发者自定义角色回复。


详细配置说明请参见[回复角色配置](/reference/app-config/reply-role)文档。
