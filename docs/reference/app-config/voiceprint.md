## 背景

在智能交互系统中，用户追求极致的个性化体验。若交互系统能识别用户身份，即可调用用户专有偏好来提高任务执行效率，建立模拟人类的情感记忆，并实现公用设备的用户隐私隔离。

## 声纹识别介绍

声纹是说话人语音中蕴含的生物特征，可用于身份标识。**声纹识别**指在用户说话时，AIUI 系统实时分析语音特征，与设备注册的声纹数据库比对，给出声纹匹配的个性化用户信息，判断"说话人是谁"。

::: tip 温馨提示
1. AIUI 声纹识别服务与设备强绑定，具体使用方式见[声纹管理 API](/api-dev/ultra-chain/voiceprint-api)。
2. 声纹能力除绑定识别结果外，还参与长时记忆处理：单设备下不同用户声纹会单独存储一份长时记忆数据。详见[长时记忆配置](/reference/app-config/long-term-memory)。
:::

### 功能特性

| 特性 | 说明 |
| --- | --- |
| 用户主动注册声纹 | 提供声纹注册接口，支持通过声音为终端用户注册唯一标识（`member_id`），支持填写特征描述文本 |
| 一个用户多份声音 | 支持一个用户上传 3 份声音资源，以提高声纹识别准确率 |
| 设备级声纹隔离 | 不同终端设备注册的声纹库相互隔离 |
| 声纹管理 | 支持声纹的增删查改，提供管理接口 |
| 声纹信息 | 包括 `member_id`、特征描述文本、性别、年龄 |

## 使用方式

::: tip 温馨提示
声纹识别服务能力仅在 AIUI 极速超拟人交互链路下支持。如需识别匹配，还需配置最新的**通用-中文-方言融合**识别引擎。
:::

1. 联系讯飞商务申请获取能力授权
2. 授权通过后，按照[声纹管理 API](/api-dev/ultra-chain/voiceprint-api) 说明注册声纹信息
3. 在 AIUI 极速类应用管理页面 **语义模型配置 > 个性化设置** 下勾选开启声纹识别即可
