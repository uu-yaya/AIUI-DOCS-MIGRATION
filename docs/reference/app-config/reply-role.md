## 概述

回复角色配置是 AIUI 极速超拟人交互类应用提供的配置项目。该配置项展示极速链路当前在基础配置中配置的回复角色信息，并提供全链路语音合成服务开关和发音人选配。

## 回复角色展示

在基础配置中选择对应角色（官方或自定义）后，此处将展示该角色的基础属性信息。


## 合成开关

角色发音人开关代表全链路语义后合成能力开关。打开后，每次对话 AIUI 云服务将大模型回复内容根据选配的发音人进行云端合成，下发合成音频数据。


## 音色更换

在发音人图标后点击"更换"，可跳转到发音人选择窗口。发音人分两大类：

### 官方发音人

可通过以下维度筛选：

| 筛选维度 | 可选值 |
| --- | --- |
| 年龄段 | 成人女、成人男、儿童女、儿童男 |
| 类型 | 普通、超拟人、极速超拟人 |

::: info 说明
极速超拟人与超拟人发音人的区别在于，极速超拟人发音人支持流式合成（单次合成时，合成文本支持分段流式发送）。
:::

### 声音复刻

声音复刻类型发音人可在回复角色创建页面中自定义生成。


### VCN 取值

开发者在选择发音人时，可通过发音人图标右上角查看当前发音人的 VCN 取值，端侧主动合成时可直接使用。

### 授权状态

发音人图标上的"授"字图标代表授权信息：

| 颜色 | 含义 |
| --- | --- |
| 蓝色 | 已授权，可直接选配使用 |
| 橙色 | 未授权，需联系讯飞授权后使用 |

::: tip 温馨提示
AIUI 极速交互类应用免费全开放了超拟人和极速超拟人发音人，开发者可直接免费选配使用。
:::


## 发音人试听

提供在线试听发音人音色的功能，支持两种方式：

- 在发音人选择界面直接点击播放按钮试听
- 选择发音人后自定义合成文本、调节合成参数进行合成试听
