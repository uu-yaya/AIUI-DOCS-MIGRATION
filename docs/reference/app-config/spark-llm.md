## 概述

该配置项是传统应用开启大模型能力的开关，配置项的处理关系到两个服务链路的切换：

| 开关状态 | 服务链路 | 说明 |
| --- | --- | --- |
| 开启 | 通用大模型交互链路 | 支持文档问答、旧版智能体、联网搜索、知识溯源配置 |
| 关闭 | 传统语义链路 | 不支持大模型服务请求 |

## 配置说明

开关大模型配置时，平台将弹窗提示当前操作对应的 AIUI 服务链路变更。开发者做 AIUI 集成开发时，需重点关注端侧协议的切换和结果解析。

::: tip 温馨提示
星火大模型配置关闭后，下方配置项即使已配置开启，能力也将不再生效。因为此时 AIUI 服务切换到传统语义链路，不再支持大模型相关能力。
:::


## 问答

该配置项用于为通用大模型服务链路配置自定义的**文档问答**。

### 文档问答配置

点击"添加"，可配置当前账号下已创建并发布的文档问答。


### 文档问答设置

点击"设置"，可对文档问答生效策略进行配置（针对所有文档问答生效）：

| 配置项 | 可选值 | 说明 |
| --- | --- | --- |
| 检索方法 | 单向量召回 / 多路召回 | 计算知识片段与用户问题相关性的方法 |
| 相关性阈值 | 0 ~ 1 | 低于阈值的知识片段将被排除 |

::: info 知识说明
- **相关性**：知识片段和用户问题的匹配程度，数值在 0 到 1 之间，值越高匹配度越好。
- **检索方法**：单向量召回依靠语义向量相似度计算；多路召回将语义向量、关键词等多种方式的相似度合并计算。
- **相关性阈值**：例如阈值设为 0.6，则只保留相关性 ≥ 0.6 的知识片段，避免不相关知识干扰回答。
:::


## 智能体

该配置项用于为通用大模型服务链路配置自定义的**旧版智能体**。

点击"添加"，可配置当前账号下已创建并发布的智能体；点击"创建智能体"可跳转到智能体开发界面。

::: info 知识说明
- **旧版智能体**：仅提供意图匹配能力，开发者编辑时仅提供语料设置。
- **新版智能体**：除意图匹配能力外，开发者可按业务需要进行节点能力模块自定义编辑，与目前主流智能体开发方式基本一致。
:::


## 搜索引擎

提供大模型联网搜索能力项，默认关闭。

开启后，用户进行语音交互时，当服务判断需要搜索，会调用联网服务进行信息搜索，然后由大模型对搜索信息进行总结答复。


## 知识溯源

该能力项控制大模型服务结果的下发。通常作为服务链路中加载文档问答或联网搜索获取的知识点，内容较多，默认不下发。

开启后，AIUI 服务在交互时会新增下发 `cbm_knowledge` 结果，开发者可据此了解大模型回复内容的来源。
