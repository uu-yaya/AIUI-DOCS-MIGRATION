title: 语音合成配置
description: AIUI 传统语义链路或通用大模型链路下的全链路语音合成服务配置。
---

## 概述

语音合成配置项提供 AIUI 传统语义链路或通用大模型链路下的全链路语音合成服务开关和发音人选配。开启后，服务请求时 AIUI 云服务直接采用交互回复内容进行播报。

不同链路表现如下：

| 链路类型 | 合成行为 |
| --- | --- |
| 传统语义交互链路（未开启后处理） | 直接调用技能结果中 `answer.text` 字段取值合成 |
| 传统语义交互链路（已开启后处理） | 直接调用后处理结果中 `answer.text` 字段取值合成 |
| 通用大模型交互链路 | 采用大模型结果的回复语进行合成（由于大模型结果流式下发，云端采用分段合成） |


## 音色更换

在发音人下拉列表中选择需要的发音人，支持通过筛选条件进行筛选：

| 筛选维度 | 可选值 |
| --- | --- |
| 场景 | 全部场景、通用场景、方言场景、多语种场景、民族语场景 |
| 年龄段 | 全部年龄段、女童声、男童声、成人女声、成人男声 |

::: warning 注意
超拟人发音人仅在"全部场景"筛选条件下可见。
:::


## 发音人授权查看

在发音人下拉列表中，通过颜色区分授权状态：

| 显示状态 | 含义 |
| --- | --- |
| 黑色可选 | 已授权（永久免费发音人或付费发音人） |
| 灰色不可选 | 未授权，展示授权申请方式 |

::: warning 注意
付费发音人虽然显示为可选，但需查看真实授权的截止日期，可选不一定代表可用。
:::


## 发音人试听

提供在线试听发音人音色的功能，支持两种方式：

- 在发音人选择界面直接点击播放按钮试听
- 选择发音人后自定义合成文本、调节合成参数进行合成试听

::: warning 注意
自定义合成试听方式仅支持已授权的可选发音人。
:::
