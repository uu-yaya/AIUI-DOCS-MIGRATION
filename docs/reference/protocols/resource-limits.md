---
title: 技能资源限制
description: AIUI 技能工作室中技能、问答库、文档问答及实体的创建资源数量限制说明
---

以下关于 AIUI 技能工作室技能或问答库创建资源限制说明，针对的都是**单个 AIUI 平台账号**，而不是某个 AIUI 应用。

## 自定义技能限制

| 项目 | 限制 |
| --- | --- |
| 技能个数 | 20 个 |
| 一个技能下的意图数 | 40 个 |
| 一个意图下的提问语料数 | 500 个 |
| 一个技能下的槽位数 | 60 个 |
| 一个意图下的槽位数 | 40 个 |
| 技能导入/导出次数 | 导入：10 次/天 |

## 问答库限制

| 项目 | 限制 |
| --- | --- |
| 问答库个数 | 40 个 |
| 一个问答对问题数 | 40 个 |
| 一个问答对答案数 | 20 个 |
| 一个问答库下的问答数 | 20000 个问答对或 50000 个问题 |
| 问答库导入的文件大小/导入次数 | 文件大小：< 5M，导入次数：30 次/天 |

## 文档问答限制

| 项目 | 限制 |
| --- | --- |
| 文档问答个数 | 10 个 |
| 一个文档问答支持上传的文件个数 | 10 个 |
| 单个文件大小 | < 30M |
| 单个文件最大可拆分知识点数 | 3400 个 |

## 实体限制

| 项目 | 限制 |
| --- | --- |
| 实体个数 | 40 个 |
| 一个词条类实体下的词条数 | 20000 个 |
| 实体导入的文件大小/导入次数 | < 5M，30 次/天 |
| 一个动态实体下的资源数 | 5 个 |

:::tip
签约客户可联系[技术支持](https://aiui.xfyun.cn/ask/submit)，免费提额。
:::
