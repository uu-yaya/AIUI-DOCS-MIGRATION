## 概述

错误码是系统反馈问题的重要标识，了解不同类型的错误码有助于快速排查问题。

::: info 说明
若错误码在本文中查询不到，且使用了唤醒、合成等能力，可前往 [MSC 错误码列表](https://www.xfyun.cn/document/error-code) 查询。
:::

## 大模型服务错误码

### 链接建立相关

| 错误码 | 含义 | 处理方式 |
| --- | --- | --- |
| 401 | 参数校验失败 | 签名验证失败。检查：1) `api_key` 和 `api_secret` 是否正确；2) `host`、`date`、`request-line` 是否按协议拼接；3) Signature 的 Base64 长度是否为 44 字节 |
| 403 | 时间检验失败 | `date` 参数时间戳格式不对或与服务端偏差超过 300 秒 |
