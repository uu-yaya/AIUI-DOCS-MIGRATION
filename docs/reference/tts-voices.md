## 概述

AIUI 平台目前提供的合成发音人分三大类：

| 类型 | 说明 | 支持链路 |
| --- | --- | --- |
| 极速超拟人发音人 | 支持流式文本合成的超拟人发音人 | 极速超拟人交互链路 |
| 超拟人发音人 | 精准模拟人类发言的呼吸、叹气、语速变化等，发音更自然 | 通用大模型和极速超拟人交互链路 |
| 普通发音人 | 传统发音人 | AIUI 全服务链路 |
