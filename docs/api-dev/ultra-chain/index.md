---

::: info 概述
极速超拟人链路是 AIUI 面向沉浸式交互场景全新推出的服务链路，在大模型能力基础上进一步融合 **低延迟端到端语音交互、情感表达、多模态感知** 能力，让设备的对话"听得更快、答得更像人"。适合虚拟人、智能玩伴、车载语音助手、教育伴读等对实时性与拟人化要求极高的场景。
:::

## 适用场景

- 虚拟数字人、虚拟偶像的实时语音驱动
- 儿童玩伴、情感陪伴机器人
- 车载语音助手（低延迟是关键）
- 教育伴读、口语陪练
- 需要"说人话"而非"读文本"的播报场景

## 核心特点

- **毫秒级响应**：端到端极低首包延迟，对话像真人一样自然
- **超拟人合成**：结合情感、语气、停顿、呼吸声，合成效果远超传统 TTS
- **多模态输入**：支持音频 + 文本 + 图像的融合理解
- **打断与全双工**：支持用户随时打断 AI 并无缝衔接新对话
- **情感表达**：回复带有情绪标签，可驱动面部表情、虚拟形象变化
- **声纹识别**：内置声纹 API，识别"谁在说话"，支持多用户隐私隔离
