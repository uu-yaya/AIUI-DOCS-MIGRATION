title: 通用大模型链路
---

::: info 概述
通用大模型链路接入讯飞 **星火大模型**，让设备具备开放式对话、知识问答、逻辑推理、内容生成等智能能力。相比传统语义，大模型链路不受意图/语料约束，能应对自由对话，适合客服机器人、智能助手、教育陪伴、多轮复杂问答等场景。
:::

## 适用场景

- 开放式对话（聊天伴侣、情感陪伴）
- 知识问答与百科查询（百科、教育、专业领域 FAQ）
- 内容创作（写诗、总结、翻译、改写）
- 多轮复杂任务（旅行规划、购物推荐）
- 结合文档问答、智能体的垂直场景落地

## 核心特点

- **自由对话**：支持任意话题、多轮上下文记忆
- **流式响应**：边生成边下发，交互延迟更低
- **知识溯源**：可开启知识点下发，便于答案可解释
- **智能体编排**：支持文档问答、外挂插件、联网搜索等能力组合
- **个性化 + 声音复刻**：大模型回复可融合用户画像与自定义音色
