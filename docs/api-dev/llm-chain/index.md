title: 通用大模型链路
---

集成大模型交互能力的核心API与服务接口


<div class="nav-cards">

- **[服务鉴权](/api-dev/llm-chain/auth)** — 提供API访问的身份验证与授权机制，包括密钥管理、令牌生成与权限控制。
- **[交互API](/api-dev/llm-chain/interact-api)** — 提供与大模型的核心交互能力，支持多轮对话、上下文管理与流式响应。
- **[用户个性化API](/api-dev/llm-chain/personalization-api)** — 基于用户画像与历史交互数据，提供定制化的模型响应与服务体验。
- **[声音复刻API](/api-dev/llm-chain/voice-clone-api)** — 支持语音特征提取与声音克隆，实现基于特定音色的模型语音合成功能。
- **[合成能力使用](/api-dev/llm-chain/tts-usage)** — 提供丰富的语音合成功能，支持多种音色选择、语速调节及情感化语音生成。

</div>
