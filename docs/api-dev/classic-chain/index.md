title: 传统语义链路
---

::: info 概述
传统语义链路是 AIUI 最早、最成熟的交互方案，基于 **规则 + 统计** 的语义理解引擎，面向指令明确、领域固定的场景（如智能家居控制、车载导航、信息查询等）。开发者通过云端 API 上传音频并接收识别与语义结果，具备稳定、低资源消耗的特点。
:::

## 适用场景

- 指令式交互（"打开空调"、"播放周杰伦的歌"）
- 问答型查询（"今天天气怎么样"）
- 垂直领域的业务办理（订票、点餐、物流查询）
- 需要精确意图匹配与实体提取的场景

## 核心特点

- **稳定可靠**：基于规则引擎，响应可预测、结果可控
- **结构化结果**：返回标准的 intent + slots 结构，直接驱动业务逻辑
- **离线能力**：部分命令词可支持离线识别
- **个性化支持**：可上传动态实体、用户词库实现个性化识别
