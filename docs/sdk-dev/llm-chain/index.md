---

::: info 概述
通用大模型链路接入讯飞 **星火大模型**，让设备具备开放式对话、知识问答、逻辑推理、内容生成等能力。在客户端 SDK 层面，开发者可以直接获得多轮对话、流式响应、超拟人合成、声音复刻等高级能力，快速打造具备"智商"的智能产品。
:::

## 适用场景

- 开放式对话（聊天伴侣、情感陪伴机器人）
- 知识问答与百科查询
- 内容创作（写作助手、翻译改写）
- 多轮复杂任务规划（旅行、购物、学习）
- 需要结合文档问答、智能体的垂直场景

## 核心能力

- **自由对话**：支持任意话题、多轮上下文记忆
- **流式响应**：边生成边下发，首字响应低于 1 秒
- **超拟人 TTS**：配合大模型回复文本输出带情感的拟人合成音频
- **声音复刻**：支持用自定义音色播报大模型回复
- **个性化**：上传用户画像，让大模型输出更贴近用户偏好
