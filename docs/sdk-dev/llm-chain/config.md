---
title: 通用大模型链路配置说明
---

::: info 概述
通用大模型交互链路服务应用可以在6.6及以上版本SDK使用，在5.6版本及以下版本中无法使用。
:::

## SDK配置说明

通用大模型交互链路配置主要区分AIUI SDK版本：

- 5.6.xxx 及以下版本
  这些版本AIUI SDK仅支持传统语义交互链路服务，`不支持`通用大模型交互链路服务；
- 6.6.xxx 及以上版本
  这些版本中，AIUI SDK开始支持除传统语义交互链路服务外，还支持通用大模型交互服务。由于不同的服务云端服务入口不一致，所以需要在aiui.cfg中通过执行 aiui\_ver 来确定SDK数据请求地址。

```json
{
    "global":{
        "scene":"main_box",
        "clean_dialog_history":"auto",
        /* 1：传统语义 、2：通用大模型*/
        "aiui_ver": "2"
    },
}
```

- 6.8.xxx 及以上版本
  这些版本中，AIUI SDK支持除传统语义交互链路服务、通用大模型交互服务、极速超拟人交互服务。由于不同的服务云端服务入口不一致，所以需要在aiui.cfg中通过执行 aiui\_ver 来确定SDK数据请求地址。

```json
{
    "global":{
        "scene":"main_box",
        "clean_dialog_history":"auto",
        /* 1：传统语义 、2：通用大模型、3：极速超拟人*/
        "aiui_ver": "2"
    },
}
```

## AIUI应用配置说明

通用大模型交互链路使用，是需要应用配置与SDK配置共同配置使用的。

- 支持`星火大模型配置`项开关操作的应用
  开启 星火大模型配置 项，示例如下：

![](/media/202509/2025-09-12_135611_0016040.6525057605353878.png)

- 没有`星火大模型配置`项的应用
  该类型应用为极速超拟人交互类应用，应用已固定限制只能按照极速超拟人服务对接，`不支持`通用大模型服务切换对接。如确实需要对接，可以联系讯飞技术同事，进行应用信息回退。
