## 服务说明

`通用大模型交互链路`下，关于个性化数据使用与`极速超拟人交互链路`下**接口调用一致**。
详见文档[个性化数据使用](/sdk-dev/ultra-chain/personalization "个性化数据使用")说明。

## 接口说明

通用大模型链路的个性化数据接口与极速超拟人链路完全一致，包括：

- **个性化数据上传**：通过 `CMD_SYNC` 消息上传用户词表、动态实体等个性化资源
- **个性化数据下载**：查询已上传的个性化数据内容
- **个性化数据删除**：清除指定的个性化资源
- **生效使用配置**：在交互参数中配置 `pers_param` 使个性化数据生效

具体接口调用方式、参数格式及代码示例，请参阅[极速超拟人链路个性化数据使用](/sdk-dev/ultra-chain/personalization)文档。
