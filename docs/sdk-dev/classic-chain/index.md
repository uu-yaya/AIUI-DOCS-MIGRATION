::: info 概述
传统语义链路是 AIUI SDK 最成熟稳定的接入方案，基于 **规则 + 统计** 的语义理解引擎。适合指令明确、领域固定的场景（智能家居、车载导航、信息查询等），在移动端与嵌入式硬件上具备低资源消耗、结果可预测的特点。
:::

## 适用场景

- 指令式交互（"打开空调"、"调高音量"）
- 垂直领域业务办理（订票、点餐、物流查询）
- 精确意图匹配与实体提取的智能家居、车载控制
- 对识别结果稳定性和可控性要求高的嵌入式场景

## 核心能力

- **结构化语义结果**：返回 intent + slots 格式，直接驱动业务逻辑
- **热词与动态实体**：通过个性化数据上传提升特定词汇识别率
- **离线唤醒 + 在线识别**：本地唤醒后触发云端语义理解
- **会话上下文**：支持多轮对话槽位填充
