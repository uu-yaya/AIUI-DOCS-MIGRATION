## 合成发音人说明

AIUI平台目前提供的合成发音人分三大类：

- 极速超拟人发音人
   支持流式文本合成的超拟人发音人
- 超拟人发音人
   在传统语音合成基础上，精准模拟人类发言的呼吸、叹气、语速变化等，使得发音更自然的发音人
- 普通发音人
   传统的发音人
