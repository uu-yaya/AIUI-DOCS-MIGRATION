## AIUI SDK基础能力

探索SDK提供的核心功能与特色能力

▼

### 流式识别

实时处理语音流并转换为文本，支持边说边识别的连续交互模式。

[查看详情](/sdk-dev/features/streaming-asr)
### 离线识别

无需网络连接即可进行语音识别，保障无网络环境下的基本功能使用。

[查看详情](/sdk-dev/features/offline-asr)
### 语音唤醒

通过特定唤醒词激活设备，支持自定义唤醒词及灵敏度调节。

[查看详情](/sdk-dev/features/wakeword)
### 语音合成

将文本转换为自然流畅的语音，支持多种音色、语速和音量调节。

[查看详情](/sdk-dev/features/tts)
### 用户个性化

根据用户习惯和历史数据提供个性化交互体验，提升识别准确率。

[查看详情](/sdk-dev/features/personalization)
### 自定义参数

支持通过参数配置定制SDK行为，满足不同场景下的个性化需求。

[查看详情](/sdk-dev/features/custom-params)
