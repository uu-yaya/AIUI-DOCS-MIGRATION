AIUI SDK 是讯飞为客户端开发者提供的完整语音交互库，封装了麦克风采集、VAD 端点检测、网络通信、鉴权、结果解析等底层能力，让开发者专注于业务逻辑，快速在 Android / iOS / Windows / Linux 等平台上实现语音交互功能。
:::

## 什么是 AIUI SDK

AIUI SDK 是讯飞开放平台发布的一套客户端软件开发包，内置了与 AIUI 云端服务通信的完整协议栈。开发者只需在项目中引入 SDK 文件，调用简洁的 API 即可完成语音唤醒、识别、理解、合成的全链路对接，无需处理底层细节。

- **面向人群**：Android / iOS App 开发者、Windows/Linux 桌面应用开发者、嵌入式硬件开发者
- **集成方式**：下载平台提供的 SDK 包，按平台规范引入到工程中
- **核心价值**：屏蔽协议差异、提供离线能力、统一多端开发体验

## SDK 与 API 的区别

| 维度 | SDK | API |
|------|-----|-----|
| 集成方式 | 在应用中集成讯飞提供的客户端库 | 直接发 HTTP/WebSocket 请求 |
| 适用平台 | Android / iOS / Windows / Linux | 任意（服务端、Web、嵌入式） |
| 开发语言 | 依 SDK 平台（Java / Swift / C++ 等） | 任意（Python、Node.js、Java、Go 等） |
| 特色能力 | 支持离线唤醒/识别、本地缓存 | 纯云端，依赖网络 |
| 典型场景 | 移动 App、智能硬件、桌面程序 | 服务端中转、Web 应用、跨平台工具 |

## 核心能力

AIUI SDK 提供了以下核心能力模块：

- **语音交互**：麦克风采集 + VAD + 识别 + 语义 + 合成的完整链路
- **唤醒词**：本地离线语音唤醒，毫秒级响应，支持自定义唤醒词
- **流式识别**：边说边返回识别结果，降低交互延迟
- **个性化**：热词、用户词典、动态实体等个性化数据的上传与使用
- **多链路支持**：同一 SDK 可切换传统语义、通用大模型、极速超拟人三条链路
