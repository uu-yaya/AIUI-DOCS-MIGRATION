## 大模型服务相关错误码

> 链接建立相关

| 错误码 | 含义 | 处理方式 |
| --- | --- | --- |
| 401 | 参数校验失败 | 签名验证失败，可能原因有很多：1. 检查api\_key,API\_secret 是否正确 2.检查计算签名的参数host，date，request-line是否按照协议要求拼接。 3. 检查signature签名的base64⻓度是否正常，正常44个字节，如果超出此范围，则可能是参与base64计算的签名信息已经过编码了,需要使⽤原始字节进⾏base64编码。 |
| 403 | 时间检验失败 | date参数时间戳格式不对或者时间已经过期，与服务端时间戳偏差超过了300s，请检查传⼊的date参数 |
