::: info 概述
极速超拟人链路是 AIUI 面向沉浸式交互场景推出的 SDK 接入方案，在大模型能力基础上融合 **端到端低延迟、情感表达、多模态感知、RTOS 适配** 等特性。面向虚拟人、智能玩伴、车载助手、教育陪伴等对实时性与拟人化要求极高的硬件产品。
:::

## 适用场景

- 虚拟数字人、虚拟偶像的实时语音驱动
- 儿童玩伴、情感陪伴机器人
- 车载语音助手（低延迟是关键）
- 教育伴读、口语陪练
- 资源受限的 RTOS 嵌入式设备

## 核心能力

- **毫秒级响应**：端到端极低首包延迟，对话像真人一样自然
- **超拟人 TTS**：结合情感、语气、停顿、呼吸声，合成效果远超传统 TTS
- **流式合成**：边生成边播放，进一步降低听感延迟
- **声音复刻**：少量样本即可生成专属品牌音色
- **情感表达**：回复带有情绪标签，可驱动面部表情、虚拟形象变化
- **RTOS 适配**：为低内存、低算力嵌入式硬件提供专用轻量 SDK
