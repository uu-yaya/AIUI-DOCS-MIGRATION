---
title: 极速超拟人链路配置说明
---

::: info 概述
极速超拟人交互链路**只能在6.8及以上版本中使用**，在6.6及以下版本中无法使用。>只能在6.8及以上版本中使用，在6.6及以下版本中无法使用。
:::

## SDK配置说明

极速超拟人交互链路配置主要区分AIUI SDK版本：

- 5.6.xxx 及以下版本
  这些版本AIUI SDK仅支持传统语义交互链路服务，`不支持`极速超拟人交互链路服务；
- 6.6.xxx 版本
  这些版本中，AIUI SDK支持传统语义交互链路服务、通用大模型交互链路服务，`不支持`极速超拟人交互链路服务；
- 6.8.xxx 及以上版本
  这些版本中，AIUI SDK支持除传统语义交互链路服务、通用大模型交互服务、极速超拟人交互服务。由于不同的服务云端服务入口不一致，所以需要在aiui.cfg中通过执行 aiui\_ver 来确定SDK数据请求地址。

```json
{
    "global":{
        "scene":"main_box",
        "clean_dialog_history":"auto",
        /* 1：传统语义 、2：通用大模型、3：极速超拟人*/
        "aiui_ver": "3"
    },
    // v3版本头部参数
    // "prot_interact_mode" 交互协议，跟speech中的interact_mode没有关联，v3才支持。
    // 取值："oneshot"（使用本地vad时推荐，默认取值，可以支持interact_mode设置单工或双工交互模式）
    // 取值："continuous"（使用云端vad时推荐，注意关闭本地vad配置并配置interact_mode取值为双工交互模式）
    "header": {
        "prot_interact_mode": "oneshot"
    },
}
```

## AIUI应用配置说明

极速超拟人交互链路服务使用，只能在没有星火大模型配置项的应用中对接。

- 支持`星火大模型配置`项开关操作的应用，`不支持`极速超拟人交互链路服务；若想体验极速交互链路服务，新建应用即默认为极速超拟人交互类应用。
- 没有`星火大模型配置`项的应用
  该类型应用默认为极速超拟人交互类应用，**极速超拟人服务只能在该类应用中对接**，云端无需做任何操作。
